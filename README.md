## ▶️ Como executar o projeto

### 1 Iniciar o servidor (Frontend)

Execute o comando abaixo no terminal para iniciar a aplicação e acessar a interface web:

node server.js

No navegador acesse: 

http://localhost:3000

### 2 Iniciar o servidor (Backend)

Execute o comando abaixo no terminal para iniciar os testes de Backend:

node testes.js
